There are 3 types of Lists:

definition lists
which show terms and definitions
unordered lists
which show bullets
ordered lists
which show numbers or bullets

Above is an example of a definition list.

Unordered list can have a type attribute
which can be applied to the UL or the LI tag.
The three values allowed are:

  • Type = "square"
  • Type = "disc"
  • Type = "circle"

Ordered lists can have a type too.
Used along with a start attribute they can show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbers
